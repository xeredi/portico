FICHEROS EUROSTAT
-----------------

Nombre Fichero: MRTM_<tipoFichero>_Q_ES_<año>_000<trimestre>.TXT

Campos separados por ';'

Capos fijos
	- Tipo Fichero: A1, A2, ...
	- Año
	- Trimestre
	- Puerto Declarante

Campos variables en funcion del tipo de fichero
	- c0 : Dirección transporte
	- c1 : Pais/Pto carga/descarga
	- c2 : Zona costera
	- c3 : Unidad de carga
	- c4 : Tipo cargamento
	- c5 : Registro buque
	- c6 : Tipo buque
	- c7 : Tamaño buque
	- c8 : Tamaño buque GT
	- c9 : Toneladas
	- c10: Número pasajeros
	- c11: Número unidades llenas
	- c12: Número unidades vacías
	- c13: Número de buques
	- c14: Peso muerto
	- c15: Gts
	- c16: Numero de cruceros
	- c17: Numero de cruceros en tránsito
